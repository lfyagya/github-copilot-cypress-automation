## Documentation Paths

### Learning Path 1: Beginner - Getting Started
Start here if you're new to Cypress automation with Copilot:
1. Read the [Session Flow Guide](SESSION_FLOW.md)
2. Review basic [Prompts Reference](PROMPTS.md)
3. Explore practical examples in the repository

### Learning Path 2: Intermediate - Architecture & Design
Deepen your understanding of the system design:
1. Study the [Architecture Documentation](docs/ARCHITECTURE.md)
2. Review advanced [Prompts Reference](PROMPTS.md)
3. Understand integration patterns and best practices

### Learning Path 3: Advanced - Custom Implementations
Build custom automation solutions:
1. Analyze the [Architecture Documentation](docs/ARCHITECTURE.md)
2. Reference the [Session Flow Guide](SESSION_FLOW.md) for complex workflows
3. Implement custom prompts based on the [Prompts Reference](PROMPTS.md)

### Learning Path 4: Best Practices - Production Ready
Prepare automation for production environments:
1. Complete all previous learning paths
2. Review architecture patterns in [Architecture Documentation](docs/ARCHITECTURE.md)
3. Implement recommended session flows from [Session Flow Guide](SESSION_FLOW.md)
4. Use optimized prompts from [Prompts Reference](PROMPTS.md)

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- Cypress
- GitHub Copilot extension

### Installation
```bash
npm install
```

### Running Tests
```bash
npx cypress open
```
